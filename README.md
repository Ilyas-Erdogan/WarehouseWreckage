# Warehouse Wreckage
Latest Release: v1.0.0

Warehouse wreckage is a simple game where the objective is in the title: wreck the warehouse! This game was made using blueprints while following an Unreal Engine 5 course.

___

## Controls:  
Look Around: Mouse  
Move: WASD  
Launch: Space
